# 13. Colors and Backgrounds

This exercise showcases standard color utilities used to build operational status dashboards.

## Exercise 13.1: Contextual Colors and Themes
Applied standard contextual backgrounds to represent different operation states:
- **`bg-primary` & `text-white`**: Informative blue (Network).
- **`bg-success` & `text-white`**: Active/Successful green (Database status).
- **`bg-warning` & `text-dark`**: Warning yellow (Disk space alert).
- **`bg-danger` & `text-white`**: Critical error red (Node outages).
- **`bg-info` & `text-dark`**: Standard system info teal.
- **`bg-secondary` & `text-white`**: Neutral backup state gray.

## Exercise 13.2: Dark Gradient Headers
- **`bg-dark`**: Background color set to Bootstrap's dark theme slate.
- **`bg-gradient`**: Overlays a linear gradient that fades slightly from top-left to bottom-right, giving the dark backdrop a clean depth.
