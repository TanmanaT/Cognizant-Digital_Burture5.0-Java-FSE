# 7. Typography

This section presents the full spectrum of typography customization utilities in Bootstrap 5.

## Exercise 7.1: Headers and Weights
- `display-1` to `display-6`: Large headlines that are styled with a lighter font-weight and larger line height.
- `lead`: Makes a paragraph stand out by increasing its font size and line-height slightly.
- `text-muted`: Mutes text by applying an opacity layer or secondary gray color.
- `fw-bold`, `fw-semibold`, `fw-light`: Helper classes for modifying font weight.
- `fst-italic`: Helper class for italicizing text.

## Exercise 7.2: Text Case Utilities
- `text-uppercase`: Transforms text to UPPERCASE.
- `text-lowercase`: Transforms text to lowercase.
- `text-capitalize`: Capitalizes the first letter of each word.
