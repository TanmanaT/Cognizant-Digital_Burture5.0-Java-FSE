# 9. Buttons

This section showcases the extensive contextual classes and compound layout structures for buttons.

## Exercise 9.1: Button Types
- **Contextual Colors:** Uses standard `btn-[context]` classes like `btn-primary`, `btn-secondary`, etc.
- **Outline Buttons:** Uses `btn-outline-[context]` classes which omit background fills in favor of highlighted borders. The background becomes filled on mouse hover.

## Exercise 9.2: Button Groups and Toggles
- **`btn-group`**: Chains multiple buttons together seamlessly, merging borders and applying correct rounded corners.
- **`btn-check`**: Hides standard checkboxes/radios while mapping their active checked state onto adjacent `<label class="btn">` blocks for functional state styling.
