# 12. Spacing Utilities

This directory focuses on leveraging Bootstrap's margin and padding classes to create clean, legible layouts.

## Exercise 12.1: Margin and Padding Classes
Bootstrap uses the `{property}{sides}-{size}` format:
- **`property`**: `m` for margin, `p` for padding.
- **`sides`**:
  - `t` (top), `b` (bottom), `s` (start/left), `e` (end/right).
  - `x` (both horizontal sides), `y` (both vertical sides).
  - Blank: applies to all 4 sides.
- **`size`**: 0 to 5 (plus `auto`). E.g., `p-2` (8px), `mt-4` (24px), `py-5` (48px).
