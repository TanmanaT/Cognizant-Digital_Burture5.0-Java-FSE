# 8. Forms

This directory demonstrates standard Bootstrap 5 form components and premium floating label controls.

## Exercise 8.1: General Registration Form
- **`form-control`**: Applies standard border radius, resets, focus states, and transitions to inputs.
- **`input-group`**: Combines text or buttons directly alongside fields (e.g. `@` username prefix addon).
- **`form-check` & `form-check-input`**: Styles checkboxes and radios with a consistent modern look.

## Exercise 8.2: Floating Labels
- **`form-floating`**: Wraps a `<input>` and `<label>` to create an interactive animated form field.
- **Important Rule:** The `<input>` element must define a `placeholder` attribute (even if empty) and must be placed **before** the `<label>` element.
