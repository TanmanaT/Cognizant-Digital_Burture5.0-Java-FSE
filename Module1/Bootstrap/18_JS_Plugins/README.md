# 18. Bootstrap 5 JavaScript Plugins

This directory covers configuring interactive components powered by Bootstrap's bundle JavaScript.

## Exercise 18.1: Modal Popup
- Created using standard modal syntax layout structures (`modal`, `modal-dialog`, `modal-content`).
- Triggered using element attributes on buttons: `data-bs-toggle="modal"` and `data-bs-target="#demoModal"`.
- Smooth slide and fade overlays configured automatically by Bootstrap 5.

## Exercise 18.2: Accordion Component
- Leverages the `accordion` container to host multiple `accordion-item` blocks.
- Uses `data-bs-toggle="collapse"` to manage visibility states.
- Applies `data-bs-parent="#accordionExample"` to ensure only one accordion panel remains open at any given moment.
