Exercise 18.2: Collapsible Accordion Menu

Modals use data attributes like data-bs-toggle="modal" and data-bs-target="#demoModal". When clicked, Bootstrap's JavaScript automatically searches for the ID and applies active display styles with smooth fade animations.

Defining a parent target using data-bs-parent="#accordionExample" guarantees that only a single collapse item is displayed open at any given moment, collapsing previously active tabs automatically.
