# 10. Navbars and Navigation

This section details how to construct modern, responsive header menus and tab/pill panels.

## Exercise 10.1: Responsive Top Navbar
- **`navbar`**: Setup wrapper.
- **`navbar-expand-lg`**: Expands into a horizontal toolbar on large screens (desktops), collapsing behind a hamburger toggle menu button on smaller tablets and mobile screens.
- **`navbar-toggler`** & **`collapse navbar-collapse`**: Control elements that listen to user click events on mobile viewports.

## Exercise 10.2: Nav Tabs and Pills
- **`nav-tabs`**: Renders standard inline tab bars.
- **`nav-pills`**: Renders rounded button pill links instead of line-bordered tabs.
- Combined with `data-bs-toggle="tab"` or `data-bs-toggle="pill"` to switch views in corresponding `<div class="tab-content">` containers dynamically.
