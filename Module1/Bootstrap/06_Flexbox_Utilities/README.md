# 6. Responsive Flexbox Utilities

## Exercise 7.1: Flexbox Responsive Navbar

Implemented a flexible, responsive header navigation menu bar utilizing pure flex utility classes:
- **`d-flex`**: Activates flexbox.
- **`flex-column`**: Stacks navigation components vertically on small screens (mobile view).
- **`flex-md-row`**: Flattens navigation items into a horizontal row at the medium (`md`) breakpoint and larger.

## Exercise 7.2: Card Header Distribution

Created an dashboard card utilizing horizontal layout flexbox helpers:
- **`justify-content-between`**: Pushes name and subscription status badge to extreme opposite ends.
- **`align-items-center`**: Vertically centers elements along the cross-axis.
