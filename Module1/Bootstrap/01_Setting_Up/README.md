# 1. Setting Up Bootstrap 5

This directory demonstrates two distinct ways of linking and configuring Bootstrap 5 inside a web application.

## Exercises

### Exercise 1.1: CDN Integration
Linked Bootstrap 5 stylesheet and bundled script files directly from the jsDelivr CDN.
- **File:** [exercise_1.1.html](file:///d:/New%20folder/Module1_Bootstrap/01_Setting_Up/exercise_1.1.html)
- **Advantage:** Faster load times due to regional caching and reduced server resource consumption.

### Exercise 1.2: Local / NPM Files Integration
Linked downloaded/installed files from the node package manager (`npm install bootstrap`).
- **File:** [exercise_1.2.html](file:///d:/New%20folder/Module1_Bootstrap/01_Setting_Up/exercise_1.2.html)
- **Advantage:** Full offline availability and complete control over build pipelines and customizations.
