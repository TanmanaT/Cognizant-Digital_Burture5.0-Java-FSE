# 16. Positioning Utilities

This directory covers Bootstrap's absolute, fixed, and relative layout placement options.

## Exercise 16.1: Fixed Footer
- **`position-fixed`**: Pulls the element out of normal document flow and locks it relative to the viewport.
- **`bottom-0 start-0 w-100`**: Pins the locked bar to the bottom and stretches it full horizontal width.

## Exercise 16.2: Absolute Badge Overlay
- **`position-relative`**: Establishes a local coordinate boundaries context for child positioning.
- **`position-absolute`**: Positions the badge absolutely inside the closest relative parent boundary.
- **`top-0 start-100 translate-middle`**: Pins the badge exactly to the top right corner and shifts it 50% horizontally/vertically back to center it over the boundary edge.
