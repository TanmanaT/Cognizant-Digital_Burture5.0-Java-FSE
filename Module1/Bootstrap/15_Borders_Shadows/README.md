# 15. Borders, Shadows, and Rounded Corners

This directory details utilities to style borders, rounded corners, and elevation depth shadows.

## Exercise 15.1: Border Utilities on Avatar Image
- **`border`**: Activates borders on the target component.
- **`border-primary`**: Changes the border color to primary blue theme.
- **`border-3`**: Increases border thickness (options from 1 to 5).
- **`rounded-circle`**: Clips the element's corners into a 50% border radius circle.

## Exercise 15.2: Shadow Elevation and Pill Elements
- **`shadow-lg`**: Applies a soft, deep blur drop shadow to give cards height/depth.
- **`rounded-pill`**: Applies a 50rem border radius, creating elongated pill capsules for buttons or inputs.
