# 4. Column Layouts and Grid Classes

## Exercise 4.1: Sidebar and Content Layout

Created a standard dashboard/documentation page structure with a sidebar on medium devices and above:
- **Sidebar CSS Class:** `col-md-3` (takes 25% width on `md` screens and above).
- **Content CSS Class:** `col-md-9` (takes 75% width on `md` screens and above).
- Below the `md` threshold, they collapse and stack vertically.

## Exercise 4.2: Four-Column Layout

Created a four-column grid that spreads out horizontally on small screens and above:
- **CSS Class:** `col-sm-3` (4 columns of size 3 = 12 columns total).
- Below the `sm` threshold, they form two columns per row (`col-6`) or full width based on viewport width.
